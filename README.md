# Packages API

## Setup API locally

-   `cp .env.example .env` (_edit .env file to fit your needs_)
-   `npm install`
-   `npm run start`

## Generate a database

-   `npm run populate`

## Routes

-   `/establishments`
    -   `GET`
-   `/establishments/<id>`
    -   `GET`
    -   `POST`
    -   `PATCH`
    -   `DELETE`
-   `/packages`
    -   `GET`
-   `/packages/<id>`
    -   `GET`
    -   `POST`
    -   `PATCH`
    -   `DELETE`
